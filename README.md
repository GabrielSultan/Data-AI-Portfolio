# Data-AI-Portfolio

Exemples de petits projets en Data et IA réalisés dans le cadre de mes études d’ingénieur à l’ESILV. Ce portfolio comprend des travaux en Machine Learning, analyse de données et mathématiques appliquées à l’intelligence artificielle.

Ce repo a pour objectif d’être enrichi progressivement et de servir de support pour la création d’un site web permettant à chacun de consulter de manière intuitive les différents projets que j’ai réalisés au cours de mes études.
